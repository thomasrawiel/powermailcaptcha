=========
PSR-14 events
=========

.. versionadded:: 1.8.0
.. versionadded:: 2.2.0

The PSR-14 event `BeforeVerifyRequestEvent` has been added in order to manipulate the additional request options
that are sent to the Site-Verify-URL
