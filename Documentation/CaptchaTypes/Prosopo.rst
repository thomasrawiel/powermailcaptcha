============
Prosopo Procaptcha
============

.. versionadded:: 1.7.0
.. versionadded:: 2.1.0

- Register your domain to https://www.prosopo.io/
- Add sitekey and secretkey to the :ref:`Typoscript constants <typoscript>`
- In TypoScript Constants, set the captcha method to `procaptcha`
